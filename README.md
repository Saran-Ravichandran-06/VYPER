VYPER is a real-time Snake game built using React and Flask with WebSocket-based communication.  
It features dynamic grid sizes, speed stages, classic mode gameplay, and a modular game engine.

## Tech Stack
- Frontend: React (Vite)
- Backend: Flask, Flask-SocketIO
- Communication: WebSockets
- Styling: CSS Grid

## Features
- Real-time snake movement
- Multiple grid stages
- Speed-based game modes
- Classic (no time limit) gameplay
- Clean separation of frontend and backend

## Status
Actively developed and optimized for deployment.
